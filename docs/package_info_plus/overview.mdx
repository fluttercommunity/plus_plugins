## Package Info Plus Overview

This Flutter plugin provides an API for querying information about an
application package.

### Get started

#### 1. Add it to your project

```yaml {4} title="pubspec.yaml"
dependencies:
  flutter:
    sdk: flutter
  package_info_plus: "^{{ plugins.package_info_plus }}"
```

#### 2. Download dependency

```bash
$ flutter pub get
```

#### 3. Rebuild the app

```bash
$ flutter run
```
